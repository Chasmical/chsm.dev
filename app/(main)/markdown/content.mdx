## Code blocks

The website provides its own customized code blocks: with language icons ([`seti-ui`](https://github.com/jesseweed/seti-ui)), file names, line numbers, copy buttons, and absolutely ***spectacular*** syntax highlighting ([`shiki`](https://github.com/shikijs/shiki))!

## KaTeX math

The markdown here uses [`remark-math` and `rehype-katex`](https://github.com/remarkjs/remark-math) plugins to render math expressions with [$$\KaTeX$$](https://katex.org). They can be rendered inline: $$ \vec{v}=(m\vec{g} - \rho \vec{g}V) t $$, or in block form:

$$
f(x) = \int_{-\infty}^\infty
  \hat{f}(\xi)\,e^{2 \pi i \xi x}
  \,d\xi
$$



## Emojis

Regular emojis are converted into [Twemoji](https://github.com/twitter/twemoji) emojis. They can also be inserted using GitHub's `:emoji-name:` syntax, and `\`-escaped if you want to make rendering platform-specific.

## Inline CSS colors

CSS colors have a color box next to them: `#ff0000`, `#00ff00`, `#0000ff`.
Color functions work as well!: `rgb(175, 23, 187)` `hsl(360 88% 36%/0.5)`.



## Embeds

YouTube links on separate lines turn into embeds.

https://youtu.be/AzYJUVqlUCA

The `<iframe>`s are loaded on click to prevent unwanted tracking by third parties.
